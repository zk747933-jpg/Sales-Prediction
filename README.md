# 📈 Sales Prediction using Python

## 📌 Project Overview

This project aims to predict product sales based on advertising expenditure across different media channels such as **TV, Radio, and Newspaper**. The project includes data preprocessing, exploratory data analysis (EDA), visualization, and a Machine Learning model using **Linear Regression**.

---

## 🎯 Objective

To analyze the relationship between advertising budgets and product sales, and build a regression model capable of predicting future sales.

---

## 📂 Dataset

The dataset contains the following features:

- TV
- Radio
- Newspaper
- Sales (Target Variable)

---

- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

## 📈 Model Evaluation

The model was evaluated using:

- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- Root Mean Squared Error (RMSE)
- R² Score
